Volver a la página principal

1- Creación de usuario

Cómo habrás visto necesitas estar registrado en el sitio para comenzar a utilizarlo. Para ello deberás crearte tu usuario.

a- ¿Cómo crear un usuario?

Para crear un usuario se necesita hacer click en el link Nuevo usuario debajo del formulario de ingreso a la página. este link te redireccionará a otra página en la cuál se podrá observar un formulario como el siguiente:


En este formulario deberá completar todos los campos. Preste especial atención a los campos de ingreso de la contraseña ya que ésta deberá ser recordada para su posterior ingreso, como así también al campo "Email", en este se deberá ingresar un correo electrónico válido y al que se tenga acceso, ya que si fuese necesario recuperar la contraseña la misma será enviada a ésta dirección.
Una vez completados los campos se deberá ingresar el código de verificación correcto, este permite determinar al sistema que es una persona que se está registrando.
Completados todos estos datos presionar el botón "Registrar" para crear el usuario, si el usuario puede ser creado y no hay errores en el formulario automáticamente será redirigido a la página principal de su cuenta.

2- Restablecer contraseña

Si por algún motivo la contraseña es olvidada el sistema permite regenerar una nueva. Para ello debajo del formulario de ingreso en la página principal exite un link con el nombre Olvido de usuario o contraseña. Este link redireccionará a una página con un formalario como el siguiente:


Una vez ingresado el correo electrónico asociado al usuario del cuál se desea restablecer la contraseña se enviará un correo, en este se incluirá una contraseña generada al azar que luego podrá ser cambiada por el usuario en el panel de edición de perfil.

3- Plataforma de usuario

a- Ingreso al sistema

Una vez creado el usuario podrá ingresar a su cuenta a través de la página principal, completando el siguiente formulario:


Ingresando el nombre de usuario y la contraseña asociada se podrá ingresar a la página principal del usuario.

b- Partes del escritorio del usuario

Cuando el usuario ingresa por primera vez al sistema, se encontrará con una pantalla como la siguiente:


Las partes de esta pantalla se describen a continuación:
  • Barra superior de menú

    • Esta barra a la izquierda de la pantalla muestra el logo del sitio y a la derecha contiene dos partes, primero una lista de botones(que varía de acuerdo a la página en la que se esté) para realizar diferentes acciones y la segunda parte muestra el nombre del usuario, que al posicionar el mouse sobre este desplegará una menú con dos opciones, una para editar los datos del usuario y otra, para desconectarse del sistema.

  • Escritorio contenedor de los bonsáis

    • Ésta es la parte principal de la página. Allí, en donde ahora se muestra la frase "Bienvenido a su espacio. Para agregar su primer Bonsái presione el botón "Agregar bonsái" en la parte superior derecha de la página" se irán mostrando los Bonsáis cargados por el usuario.

  • Pié de página
    • Éste se encuentra, como su nombre lo indica, al final de la página. Está formado por tres componentes principales, estadísticas del sitio, que muestra algunas estadísticas generales; la licencia de uso en la parte central; y a la derecha, un link de contacto para informar cualquier fallo o hacer alguna sugerencia.

c- Editar perfil

El usuario puede editar la información cargada cuando se registró en el sistema como así también, cambiar la contraseña con la que ingresa al sitio.
Para acceder a esto, se deberá posicionar el puntero del mouse sobre el nombre de usuario en la esquina superior derecha de la pantalla y allí se desplegará un menú como el de la imagen siguiente, y allí hacer click en "Editar perfil".


Echo el click se ingresará a la pantalla de modificación de datos de usuario, que se detalla a continuación.

Los datos de un usuario que pueden ser modificados son los que muestra la imagen a continuación:


Cabe aclarar que el resto de los usuarios no ven todos los datos que coloque aquí, sólo pueden llegar a aparecer el nombre y apellido en la página principal cuando el sistema muestre uno de sus bonsáis. ii- Modificación de la contraseña
El usuario registra por primera vez su contraseña cuando se registra en el sistema, pero el sistema provee la posibilidad de renovarla cuando desee. Para ello, desde el formulario de modificación de los datos de usuario, se debe hacer click en el botón "cambiar contraseña" y el sitio lo redireccionará al formulario mostrado a continuación:


En este formulario se deberá ingresar la contraseña actual y dos veces la nueva contraseña. Esto se debe a que se verifica que la nueva contraseña ingresada no sea la no deseada. Una vez echo esto, se deberá presionar guardar y la nueva contraseña será guardada.

d- Cerrar sesión

Para el cierre de sesión de usuario se debe posicionar sobre el nombre del usuario, en la esquina superior-derecha, allí se desplegará un menú el cual se muestra en la imagen siguiente. Allí hacer click en "Cerrar sesión".


Cabe aclarar que si esta operación no se hiciese, la sesión caducará automáticamente después de los 30 minutos.

e- Agregar un bonsái

Bien, a partir de aquí comienza la parte interesante. Como se mostró en la sección 3-b, la pantalla principal contiene un botón con la leyenda "Agregar Bonsái". Al hacer click en este se abrirá la siguiente página:


En este formulario se pueden observar los siguientes campos:
  • Nombre vulgar
    • Es el nombre común de la especie que se le da en la región, por ejemplo Ombú o Pino Negro. Este campo es obligatorio.
  • Nombre botánico
    • Según Wikipedia, en botánica, el nombre botánico es el nombre asignado a cada planta. Este es dado de acuerdo a el "Código Internacional de Nomenclatura Botánica". Este campo es opcional.
  • Foto
    • Esta es la foto del Bonsái. Cabe aclarar que para agregar una foto al Bonsái no es posible hacerlo subiéndola al sitio, se deberá en cambio, subirla por ejemplo a un álbum en Facebook o en algún lugar en el cual se puedan almacenar fotos. Luego de esto, se deberá copiar la dirección de esta imagen y pegarla en este campo. A continuación se detalla como copiar la dirección de la imagen: Para copiar la dirección de la imagen se deberá abrir la misma y sobre esta, se deberá hacer click con el botón derecho y elegir la opción "Copiar ruta de la imagen", "copiar URL de la imagen", "copiar dirección de la imágen", etc. Las diferentes opciones dadas son dependiendo del navegador que se esté utilizando. Luego se deberá hacer click con el botón derecho sobre el campo Foto y seleccionar "pegar". Echo esto se podrá visualizar al costado derecho la vista previa de la foto elegida. Este campo es opcional
  • Apodo
    • Este es un campo muy particular, se agregó ya que muchas personas le ponen nombres o apodos a sus amados Bonsáis, entonces aquí podrán colocarlo. Este campo es opcional.
  • Fecha de nacimiento/adquisición(*)
    • Esta es la fecha en la que la planata nació o fué adquirida por el bonsaista. Este campo es obligatorio.
  • Estilo
    • El estilo es la forma que se le dá al árbol. Este campo es opcional.
Una vez completados los campos, se deberá presionar el botón "Guardar" y el nuevo Bonsái quedará almacenado.
